# Hansen_AS

Analysis code and imaging pipelines for the Kjer-Hansen et al. paper "Alternative splicing fine-tunes modular protein interfaces to control protein localisation"

## Repository structure

- `python_scripts/`  
  Core Python analysis scripts for network and structural analysis.

- `imaging_scripts/`  
  Image analysis and quantification scripts used for high content microscopy screen.

## Requirements

- Python ≥ 3.9
- R ≥ 4.2
- See subdirectory READMEs for package requirements.
